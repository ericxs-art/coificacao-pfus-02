TOONTUBERS

TOONTUBERS foi uma série de vídeos criada pelo Cartoon Network onde os personagens Mordecai e Rigby, apareciam como se fossem youtubers gamers. Eles jogavam e comentavam jogos populares, como se estivessem fazendo vídeos de gameplay, com muito humor e zoeira. Foi feito para parecer um canal de jogos no estilo dos youtubers reais, só que com os personagens do desenho.
